# 🌿 ESG Carbon Emission Management System

> 기업 ESG 담당자가 엑셀 중심으로 관리하던 탄소배출 데이터를  
> 자동 수집 · 산정 · 시각화하여  
> 보고서 작성 시간을 줄이는 ESG 통합 관리 플랫폼

---

## 🚀 Why This Project?

많은 기업들이 ESG 공시 의무 확대에 따라 탄소배출량 데이터를 제출해야 하지만,

- 사업장별 데이터가 흩어져 있고
- 수작업 엑셀 관리 비중이 높으며
- 산식 오류 및 휴먼에러 리스크가 존재합니다.

본 프로젝트는 이러한 문제를 해결하기 위해  
탄소배출량 데이터를 자동 계산하고 관리하는 시스템을 구축합니다.

---

## 🎯 Target Users

- 중소 · 중견기업 ESG 담당자
- 제조업 운영관리 부서
- 지속가능경영 / 경영기획 조직
- ESG 컨설팅 실무자

---

## ✨ Core Features

### 1. 데이터 업로드 및 통합 관리
- Excel / CSV 배출 데이터 업로드
- 사업장별 데이터 통합 저장

### 2. 탄소배출량 자동 산정
- Scope 1 / Scope 2 계산 로직 적용
- 배출계수 기반 자동 계산

### 3. 대시보드 시각화
- 월별 / 사업장별 배출량 현황
- 목표 대비 추이 분석

### 4. 보고서 자동화
- ESG 제출용 리포트 생성
- PDF / Excel Export

### 5. 권한 및 보안
- 사용자 인증 / 권한 분리
- 민감 데이터 암호화 저장

---
